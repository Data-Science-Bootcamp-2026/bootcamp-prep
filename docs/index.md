## Welcome

We are almost at our first ever data science bootcamp offered by the Students of the International Society for Pharmacoepidemiology (SISPE) University of Toronto (UofT) Chapter! We will be exploring **data science** topics such as programming in R, open science, dependency management, version control, and coding agents. Although this bootcamp is about data science, we will be using examples from the field of **pharmacoepidemiology**, which is the study of the use, effectiveness, and safety of drugs.

## Scope

Participants are coming from diverse backgrounds, including undergraduates, graduates, post-graduates, pharmacy students, and working professionals. No prior coding experience is required to participate, though some familiarity with coding may make it easier to follow along with the concepts.

I would like you to go through this book to **make sure you have everything set up beforehand**. All the software and services listed are free or have a free tier. Please reach out if you run into any problems, but even in the worse case we can figure it out during the bootcamp. After all, we are hosting it in person precisely to give hands-on support and troubleshooting. Nonetheless, if you cannot make it, all materials will be shared afterwards.

::: {.callout-warning}
## **Warning**
If you are new to these software and services, allow extra time to complete the setup. It may take as little as an hour. However, he longest set up -- GitHub Education (optional, though) -- requires 3 days to authenticate.

Please bring your computer to the bootcamp. If you are bringing a work computer, please try to set up everything beforehand. Otherwise, unexpected user permission restrictions may prevent some software from being installed.
:::

In this pre-bootcamp and in the actual bootcamp, our focus will be on using graphical user interfaces (GUI) as much as possible, and command-line interface (CLI) only when necessary. Although GUI is more intuitive, there are some commands (especially for installing software) that aren't available in GUI.
